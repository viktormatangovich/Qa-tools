# QA Tools

Репозиторий содержит инструменты для QA-тестирования и отладки веб-приложений.

## Состав репозитория

### [Reqpane](qa-request-inspector/README.md) — Chrome-расширение для отладки API

Расширение для Chrome, которое перехватывает и отображает все API-запросы, ошибки консоли и сетевую активность прямо в боковой панели браузера.

**Возможности:**

- Перехват `fetch` и `XMLHttpRequest` запросов
- Фильтрация по статусам, доменам, длительности
- Моки (Mock Responses) и точки остановки (Breakpoints)
- Нагрузочное тестирование
- Экспорт в HAR, Postman, Markdown
- Тёмная тема и настраиваемый интерфейс

Подробнее — в [README проекта](qa-request-inspector/README.md).

## Архивы для установки

- [`qa-tools.zip`](qa-tools.zip) — архив расширения для загрузки в Chrome
- [`qa-tools-updated.zip`](qa-tools-updated.zip) — обновлённая версия архива

### Установка расширения

1. Скачайте один из архивов (`qa-tools.zip` или `qa-tools-updated.zip`)
2. Распакуйте архив в отдельную папку
3. Откройте Chrome и перейдите на `chrome://extensions`
4. Включите «Режим разработчика»
5. Нажмите «Загрузить распакованное расширение» и выберите распакованную папку
