# QA Tools

**Reqpane** — это расширение для Chrome, которое перехватывает и отображает все API-запросы, ошибки консоли и сетевую активность прямо в боковой панели браузера. Создано для QA-инженеров и разработчиков, чтобы упростить отладку, тестирование и анализ сетевого взаимодействия веб-приложений.

## Возможности

### 📡 Перехват запросов

- Перехватывает все вызовы `fetch` и `XMLHttpRequest`
- Захватывает заголовки и тела запросов/ответов
- Отслеживает длительность запросов и коды статусов
- Перехват консольных ошибок (errors, unhandled rejections, console.error)
- Автовыбор — автоматический фокус на новых запросах

### 🔍 Фильтрация и поиск

- Фильтры: Все, Ошибки (4xx/5xx), Медленные (>1с), Ошибки консоли
- Фильтрация по шаблону URL
- Поиск по содержимому тела запроса и ответа

### 👁️ Режимы просмотра

- **Список** — хронологический список запросов
- **Таймлайн** — визуальная шкала времени с длительностями
- **Группировка** — группировка по доменам

### 🔄 Сравнение запросов

- Выберите два запроса для сравнения бок о бок
- Визуальный diff заголовков, тел и метаданных

### 🎭 Моки (Mock Responses)

- Создавайте правила моков с шаблонами URL и HTTP-методами
- Возвращайте произвольные коды статусов и тела ответов
- Включайте/отключайте правила на лету

### 🛑 Точки остановки (Breakpoints)

- Приостанавливайте запросы перед отправкой
- Продолжайте или отменяйте запросы через всплывающее окно
- Сопоставление URL по шаблону

### ⚡ Нагрузочное тестирование

- Воспроизводите любой захваченный запрос многократно (1–100 итераций)
- Настраиваемая конкурентность (1–10 параллельных запросов)
- Регулируемая задержка между пакетами и таймауты
- Прогресс в реальном времени с отслеживанием успехов/ошибок
- Детальная статистика: avg/min/max/median/p95/p99 время ответа
- Гистограмма распределения задержек
- Пропускная способность (запросов в секунду)
- Группировка и анализ ошибок
- Сохранение и просмотр истории тестов

### 🔎 Поиск полей в DOM

- Сканируйте DOM страницы, чтобы найти, где используются значения из API-ответов
- Подсвечивайте элементы, использующие определённые поля
- Помогает понять поток данных от API до интерфейса

### 📤 Экспорт

- **HAR** — стандартный формат HTTP Archive
- **Postman** — импорт напрямую в коллекции Postman
- **Claude prompt** — форматированный вывод для помощи AI в отладке
- **Markdown Report** — подробный отчёт об отладке
- **Сессии** — сохраняйте и восстанавливайте историю запросов

### ⚙️ Настройки

- Тёмная тема
- Регулируемый размер шрифта (маленький/средний/большой)
- Избранное для часто используемых запросов

## Быстрая установка (без кода)

1. **Скачайте** один из архивов:
   - [`qa-tools.zip`](qa-tools.zip)
   - [`qa-tools-updated.zip`](qa-tools-updated.zip)
2. **Распакуйте** архив в отдельную папку
3. **Откройте Chrome** и перейдите на `chrome://extensions`
4. **Включите** «Режим разработчика» (переключатель в правом верхнем углу)
5. **Нажмите** «Загрузить распакованное расширение» и выберите распакованную папку

Готово! Нажмите на иконку Reqpane на панели инструментов, чтобы начать отладку.

## Установка для разработчиков

1. Клонируйте репозиторий:
   ```bash
   git clone git@github.com:viktormatangovich/Qa-tools.git
   cd Qa-tools/qa-request-inspector
   ```
2. Установите зависимости:
   ```bash
   bun install
   ```
3. Соберите расширение:
   ```bash
   bun run build
   ```
4. Загрузите в Chrome:
   - Перейдите на `chrome://extensions`
   - Включите «Режим разработчика»
   - Нажмите «Загрузить распакованное расширение» и выберите папку `dist`

## Архитектура

```
background.js (service worker + CDP) → App.tsx (боковая панель)
content.js (content script) — только поиск/подсветка в DOM
```

- **`background.js`** — сервис-воркер. Использует `chrome.debugger` API (CDP) для перехвата сетевых запросов через Network domain, перехвата/моков через Fetch domain и захвата консольных ошибок через Runtime domain. Хранит запросы по вкладкам (Map), управляет правилами моков/breakpoint через `chrome.storage.local`. Подключает отладчик при открытии боковой панели, отключает при закрытии.
- **`content.js`** — контент-скрипт только для поиска и подсветки полей в DOM. Не участвует в перехвате сети.
- **`App.tsx`** — React-интерфейс боковой панели. Открывает порт `chrome.runtime.connect` при монтировании для подключения отладчика. Отображает запросы, управляет фильтрацией, экспортом, настройками, моками, точками остановки, сессиями.

## Конфиденциальность

Все данные остаются локально в вашем браузере. Расширение **НЕ** собирает, не хранит и не передаёт персональные данные на внешние серверы. Подробнее — в [PRIVACY.md](qa-request-inspector/PRIVACY.md).
